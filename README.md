# Password_Strength_Analyser 🔐
An AI-powered, browser-based password security analyzer built with 4 deep-scan analysis tools — runs entirely client-side with zero data transmission.

Features • Real-time breach detection • Instant entropy scoring & pattern recognition • Strong password suggestion • Zero tracking  

PST Tool is an AI-powered password intelligence tool that thinks like a security analyst, not just a checker.
Unlike traditional password meters that run basic rules, PST Agent uses an agentic AI architecture — it orchestrates multiple analysis layers, reasons over the combined results, and synthesizes a single intelligent verdict. It doesn't just tell you your password is weak. It tells you exactly why, how an attacker would exploit it, and what to do about it.
